# BC Scarf – E-Ticaret Web Sitesi

Şık ve kaliteli eşarp koleksiyonları sunan modern bir e-ticaret platformu.

## Kullanılan Teknolojiler

| Teknoloji | Açıklama |
|-----------|----------|
| **React** | Kullanıcı arayüzü geliştirme |
| **Vite** | Hızlı geliştirme ortamı ve build aracı |
| **Tailwind CSS** | Utility-first CSS framework |
| **React Router** | Sayfa yönlendirme ve navigasyon |
| **React Hooks** | Form yönetimi ve doğrulama |

## 📋 Özellikler

- 🛍 Ürün listeleme, filtreleme ve detay sayfaları
- 🛒 Sepet yönetimi ve ödeme akışı
- ❤️ Favori ürünler listesi
- 👤 Kullanıcı kayıt, giriş ve profil yönetimi
- 📱 Tam responsive tasarım (mobil, tablet, masaüstü)
- 📝 Blog sayfası
- 🧣 Eşarp bağlama rehberi
- 💌 Bülten aboneliği
- 🎨 Kombin önerileri ve lookbook galerisi

## 🚀 Kurulum

```bash
# Repoyu klonlayın
git clone https://github.com/KULLANICI_ADINIZ/bc-scarf.git

# Proje dizinine gidin
cd bc-scarf

# Bağımlılıkları yükleyin
npm install

# Geliştirme sunucusunu başlatın
npm run dev
```

Uygulama varsayılan olarak `http://localhost:5173` adresinde çalışacaktır.

## 📦 Build

```bash
# Production build
npm run build

# Build önizleme
npm run preview
```

## 📁 Proje Yapısı

```
src/
├── assets/          # Görseller ve statik dosyalar
├── components/      # Yeniden kullanılabilir UI bileşenleri

├── data/            # Ürün ve blog verileri
├── hooks/           # Özel React hook'ları
├── integrations/    # Üçüncü parti entegrasyonlar
├── lib/             # Yardımcı fonksiyonlar
├── pages/           # Sayfa bileşenleri
└── main.jsx         # Uygulama giriş noktası
```
